# Lyread
Android e-reader for volumes by Directmedia and Duden. Installation instructions [here](docs/index.md).
* Directmedia support based on [digibux](tree/digibux)
* Duden uses SQLite
* EPUB subset of [EpubSharp](../../asido/EpubSharp)
